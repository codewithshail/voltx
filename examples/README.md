# VoltX Examples

This directory contains runnable examples demonstrating VoltX features.

## Available Examples

- **chatbot** — Streaming chat with conversation memory
- **agent-app** — AI agent with tool calling (calculator, web search)
- **rag-app** — Document Q&A with embeddings and vector search

Each example can be scaffolded with `create-voltx-app`:

```bash
npx create-voltx-app my-app --template <name> --yes
```

Or browse the templates in `../templates/`.
